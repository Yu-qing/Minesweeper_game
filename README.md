完成功能 : 
* 遊戲過程中會隨機出現多個數字作為提示
* 與正版相異出現數字的位置並非完全位於選取格的附近
* 提供插旗設置方便標記炸彈可能位置
* 提供炸彈數目
* 附有計時系統
